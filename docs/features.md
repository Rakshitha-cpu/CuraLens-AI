# Features

## MVP (Minimum Viable Product)

The first version of CuraLens AI focuses on solving the core problem of helping patients understand handwritten prescriptions safely and accurately.

### 1. Prescription Upload
- Upload handwritten prescription images (JPG, PNG, JPEG)
- Secure image handling

### 2. AI Prescription Understanding
- Extract medicine names
- Extract dosage
- Extract frequency
- Extract treatment duration
- Extract food instructions (Before/After Food)

### 3. Confidence Verification
- Detect unclear medicine names
- Highlight uncertain text
- Ask the user for confirmation before proceeding

### 4. Medication Guidance
- Explain each medicine in simple language
- Explain why the medicine is prescribed
- Display dosage instructions
- Display timing instructions

### 5. Multilingual Support
- English
- Kannada
- Hindi

### 6. Medication Schedule
Generate a daily timetable showing:
- Morning medicines
- Afternoon medicines
- Night medicines

### 7. Patient Dashboard
- View uploaded prescriptions
- View medication schedule
- View medicine details

# AI Agent Features

## Vision Agent
Reads handwritten prescriptions using Gemini Vision.

## Verification Agent
Verifies low-confidence medicine names before processing.

## Medication Intelligence Agent
Explains medicines in patient-friendly language.

## Safety Agent
Checks for:
- Duplicate medicines
- Missing dosage information
- Common medication interaction warnings

## Reminder Agent
Creates reminder schedules for medicines.

# Non-Goals

To keep the project safe and realistic, CuraLens AI will NOT:

- Diagnose diseases
- Recommend new medicines
- Replace doctors or pharmacists
- Modify prescriptions
- Predict medical conditions

It is designed only to assist patients in understanding and following their prescribed treatment.
